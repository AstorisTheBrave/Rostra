## what this is

Lets members send feedback to the staff: `/feedback send` posts their message into a designated channel
(optionally anonymously). Staff pick the channel once.

## technical map

**file paths**
- command: `src/modules/feedback/index.ts`
- config: `src/modules/feedback/service.ts`
- model: `prisma/schema.prisma` → `FeedbackConfig`

**command** - `/feedback send <message>` · `/feedback setup <channel> [anonymous]` (Manage Server).

**exports (service.ts)** - `getConfig`, `upsertConfig`.

## gotchas

- The submitter's ack is ephemeral; the feedback post goes to the configured channel as a Components-V2
  container (built via `@/ui`). When `anonymous` is on, the author isn't shown.
- Requires `prisma migrate` for `FeedbackConfig`.
