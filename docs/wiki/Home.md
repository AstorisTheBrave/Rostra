# Rostra Wiki

Rostra is an all in one, open source Discord bot built to scale to millions of users. One bot replaces
moderation, security, tickets, modmail, economy, leveling, giveaways, music, starboard, and much more, with
a clean Components V2 interface and full localization.

## Highlights

- Moderation, antinuke, automod with custom rules, tickets, modmail
- Economy with a shop and buyable roles, leveling with image rank cards, giveaways with entry requirements
- Multi board starboard with blacklists, downvotes, filters, display tiers, and overrides
- Localization as a core platform: ICU message format, per recipient languages, live reloadable
- Built for scale: native sharding by default, optional hybrid clustering, a Redis control bus for zero
  restart changes
