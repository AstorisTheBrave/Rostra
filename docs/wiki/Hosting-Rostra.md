# Hosting Rostra

This is the technical track: running your own copy of Rostra, from a first install to a sharded fleet, plus
contributing to the project. If you only want to use the bot in your server, see [[Using Rostra]].

## No terminal, no problem

If you want the bot online but do not want to touch a server, skip this whole track and use the click and go
guide: [[Easy Hosting]].
