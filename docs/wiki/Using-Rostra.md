# Using Rostra

This is the track for people who run a Discord server and want to use the bot. No code, no servers to manage.
If you want to host or develop the bot instead, see [[Hosting Rostra]].

## Start here

Follow these in order and you will have a configured server in a few minutes.

1. [[Setup Guide]] - invite the bot and run the setup wizard
2. [[Commands]] - every command at a glance
3. [[FAQ]] - common questions and quick fixes

Just want it online without any technical work? See [[Easy Hosting]].

## Feature guides

Each guide walks through one area, with the real commands and settings.

- [[Moderation and Security]] - antinuke, anti raid, automod, verification, logging
- [[Engagement]] - leveling, economy, starboard, welcome, giveaways, polls
- [[Tickets and Modmail]] - support channels and DM to staff
- [[Roles and Self-Roles]] - reaction roles, voice roles, vanity roles, bulk role
- [[Feeds and Notifications]] - YouTube, Twitch, Reddit, RSS
- [[Music]] - play audio in voice
- [[Utility and Extras]] - tags, reminders, birthdays, profiles, and more
- [[Starboard]] - highlight the best messages, in depth
